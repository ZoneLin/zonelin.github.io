Proxmox VE 擴大 Ubuntu 虛擬機硬碟大小

2025.03.26
閱讀時間: 1 分鐘

調整硬碟大小

在 Proxmox VE WEB UI 中擴大 Ubuntu 虛擬機的硬碟大小

硬碟空間只能放大，不能縮小

擴充硬碟分割 (physical drive partition)

  • Ubuntu 虛擬機開機進到 bash
  • 檢視目前磁碟狀態
bash
1

sudo fdisk -l
  • 擴充硬碟分割
bash
1

growpart /dev/sda 3
  • 檢視硬碟分割
bash
1

pvdisplay
  • 擴充硬碟大小
bash
1

sudo pvresize /dev/sda3
  • 檢視硬碟分割 看是否已改
bash
1

pvdisplay

擴充邏輯分割 Logical volume

  • 檢視邏輯分割
bash
1

lvdisplay
  • 擴充邏輯分割
bash
1

sudo lvextend -l +100%FREE /dev/ubuntu-vg/ubuntu-lv
  • 檢視邏輯分割 看是否有擴充
bash
1

lvdisplay

調整 Filesystem

  • 調整檔案系統大小
bash
1

resize2fs /dev/ubuntu-vg/ubuntu-lv
  • 檢查是否調整成功
bash
1
2

fdisk -l
df -h
