Hexo 基本使用教學

2021.11.13
閱讀時間: 1 分鐘

官網
使用說明
GitHub.

常用指令

建新貼文

terminal
1

hexo new 文章名稱

詳細請看: Writing

啟動測試伺服器

Hexo 提供 Web server 功能，不須透過 Nginx 或是 apache

安裝

terminal
1

npm install hexo-server --save

啟動伺服器

terminal
1

hexo server

詳細請看: Server

產生靜態網站檔案

terminal
1

hexo generate

詳細請看: Generating

部署到網站伺服器

terminal
1

hexo deploy

產生靜態頁面後佈署
兩個指令相同

terminal
1
2

hexo generate --deploy
hexo deploy --generate

詳細請看: Deployment

增加文章分類 (Categories)

指令開啟分類功能

terminal
1

hexo new page categories

新文章增加分類，只要在新貼文的標頭加上 categories

增加文章標籤 (Tags)

指令開啟標籤功能

terminal
1

hexo new page tags

新文章增加標籤，也是在新貼文的標頭加上 tags

修改新文章預設標頭

修改 scaffolds/post.md

這樣每次新增文章就會自動添加分類及標籤
